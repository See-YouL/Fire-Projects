◆ EEPROM_ACK_Polling()

void EEPROM_ACK_Polling ( void )

ACKNOWLEDGE POLLING 对EEPROM进行确认轮询 一旦内部定时写周期开始并且EEPROM输入被禁用，可以启动确认轮询。 这涉及发送一个启动条件，后跟设备地址。 读/写位是代表所需的操作。 仅当内部写周期完成时EEPROM 会响应“0”，允许继续读取或写入序列 以上步骤称为ACKNOWLEDGE POLLING

参数
None
返回值
None
