DMA Memory-to-Memory Transfer Example.

Author
eric
Version
None
Date
2024年1月26日14:47:55

This file contains the main function for a DMA memory-to-memory transfer example. It initializes the LED, configures the DMA, waits for the DMA transfer to complete, compares the data in two arrays, and controls the LED based on the comparison result.
